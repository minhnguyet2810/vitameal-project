# Mocha App

## Overview
Mocha App is a digital meal-kit service application built with React and TypeScript. This project utilizes Vite for fast development and build processes.

## Project Structure
```
mocha-app
├── src
│   ├── main.tsx          # Entry point of the application
│   ├── App.tsx           # Main application component
│   ├── routes
│   │   └── index.tsx     # Routing configuration
│   └── types
│       └── index.ts      # TypeScript types and interfaces
├── package.json           # npm configuration
├── tsconfig.json          # TypeScript configuration
├── vite.config.ts         # Vite configuration
└── README.md              # Project documentation
```

## Installation
To get started with the Mocha App, clone the repository and install the dependencies:

```bash
git clone <repository-url>
cd mocha-app
npm install
```

## Development
To start the development server, run:

```bash
npm run dev
```

This will start the Vite development server and open the application in your default web browser.

## Building for Production
To build the application for production, run:

```bash
npm run build
```

This will compile the TypeScript files and bundle the application for deployment.

## Linting
To check for linting errors, run:

```bash
npm run lint
```
